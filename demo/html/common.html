Include Angular, a theme from Bootswatch or the default Bootstrap CSS, and trNgGrid's JS and CSS.

Angular version requirements:

  • trNgGrid < 3.1.0 : Angular 1.2.9 - 1.3.14
  • trNgGrid    3.1.0 : Angular >= 1.3.1

<link rel="stylesheet" href="//maxcdn.bootstrapcdn.com/bootswatch/3.3.0/slate/bootstrap.min.css"/>
<link rel="stylesheet" href="trNgGrid.min.css" />

<script src="//ajax.googleapis.com/ajax/libs/angularjs/1.3.1/angular.js"></script>
<script src="trNgGrid.min.js"></script>

Assign the module trNgGrid as a dependency to your module. 

angular.module("your_app_name", ["trNgGrid"]);

Apply the tr-ng-grid directive on a table element. At minimum, you also need to pass in the array of items you wish to be listed in the grid, and that is achieved by assigning the items attribute to a property on the current scope that represents the array of items.
The items will get data bound so any change in the array will get detected automatically by the grid. 


                    

                        Sorting, global filtering and per-column filtering are active by default.
